# 🌊 Aqua Ring Challenge

An interactive water ring toss game with smooth controls, moving targets, and progressive levels, built using HTML, CSS, and JavaScript.

## 🎮 Features
- Physics-based ring movement  
- Moving targets with increasing difficulty  
- Animated fish background  
- Level progression system  
- Full-screen gameplay  

## 🕹️ Controls
- ← / → : Move rings  
- Space : Push rings upward  

## ▶️ How to Run
1. Download or clone the repository  
2. Open `index.html` in your browser  

## 🛠️ Tech Stack
- HTML  
- CSS  
- JavaScript (Canvas API)
